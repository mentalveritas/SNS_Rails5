# SNS 

## 소개
SNS형태의 글/댓글 작성 및 좋아요/Follow 기능 구현

## 개발환경
- Ubuntu 16.04 LTS
- Ruby : 2.3.1
- Rails : 5.0.0.1
- Database : Dev-> sqlite3, Prod -> postgresql
- CI : Travis-CI
- Deploy : Heroku
- Additional Gems : devise, carrierwave, mini_magick, materialize-sass

## 기능
- 게시물/댓글 CRUD
- Following/좋아요 기능
- Avatar 및 게시물 이미지 업로드 기능
- 사용자 인증 및 권한관리
- 게시물/댓글 권한 관리
- Materialize를 활용한 Front-end View
